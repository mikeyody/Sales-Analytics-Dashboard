# Sales Analytics Dashboard (Power BI)

## Overview
This project is a 3-page interactive Power BI dashboard designed to analyze sales performance, revenue trends, customer behavior, and product profitability.

The dashboard simulates a real-world business intelligence solution using SQL-based data and executive-level reporting visuals.

## Tools & Technologies
- Power BI (DAX, Data Modeling, Dashboards)
- MySQL
- SQL
- CSV
- Excel (data validation)

## Dataset
- 4 relational tables: Orders, Order Items, Products, Customers
- 20+ columns
- 5,000+ rows
- Data sourced from MySQL and exported as CSV files

## Key Features
- KPI cards for Total Revenue, Total Profit, Total Orders, and Total Customers
- Time-series analysis of revenue and profit
- Product and category performance analysis
- Customer and regional insights
- Dynamic slicers for Month, Region, and Product Category
- Clean star-schema-style data model

## Skills Demonstrated
- SQL querying and relational data modeling
- DAX measure creation
- Business intelligence dashboard design
- Data visualization best practices
- Analytical problem-solving and insight generation

## How to Use
1. Download the `.pbix` file
2. Open using Power BI Desktop
3. Interact with slicers to explore trends and performance
